# simaple-web

## Development

```
yarn
yarn dev
```

## Running with server

- 본 프로그램은 python simaple server의 존재 하에서 동작합니다.
- poetry가 설치되어 있어야 합니다.

### Install and run simaple server

```bash
git clone https://github.com/simaple-team/simaple
cd simaple
poetry install
poetry run poe web
```

### Run web

```
yarn
yarn dev
```
